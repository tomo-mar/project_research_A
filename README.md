# 動画音声からの感情抽出とハイライト自動推定
本リポジトリは、ゲーム実況などの長時間の動画音声から、配信者の感情起伏を捉えてハイライトシーンを自動推定する2段階推論アーキテクチャの実装です。
LLMによる特徴量抽出と、軽量な機械学習アルゴリズムによる最適化を組み合わせることで、計算コストを抑えつつ高精度なハイライト推定を実現しています。


## フォルダ・ファイル説明
### Zero-Shot
* `Highlight_ZeroShot.ipynb`
入力された音声ファイル（テストデータは3分）を5秒ごとのチャンクに分割し、Qwenモデルによるハイライト確率を推定するスクリプト。直接推定とVADスコアを経由した推定の2通りを行う。

### Few-Shot
* `Highlight_ZeroShot.ipynb`
Zero-Shotと同様に音声をチャンク分割し、出力の見本として音声とハイライト確率のペアを記憶しているスクリプト。
VADスコア経由は棄却し、見本データの収集法（1本の連続データ/連続2チャンクからなる抽出データ/ハイライト確率が0~1まで均一に分布した抽出データ）および推定時の記憶性（無記憶/直前1チャンクの推定を記憶/以前の善チャンクの推定を記憶）を設定可能である。

### VAD
精度の問題で棄却したVAD経由でのハイライト推定に使用したプロンプトフォルダ
* `VAD_score_matrix.ipynb`
入力された音声ファイルのチャンク分割を行い、VADスコアの時系列データを出力するスクリプト。

* `highlight_estimated_VAD.ipynb`
出力されたVADスコアのCSVと、正解ラベルのCSVをマージし、ランダムフォレストを用いてハイライト確率を学習・推論するスクリプト。
