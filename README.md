# 動画音声からの感情抽出とハイライト自動推定
本リポジトリは、ゲーム実況などの長時間の動画音声から、配信者の感情起伏を捉えてハイライトシーンを自動推定する2段階推論アーキテクチャの実装です。
LLMによる特徴量抽出と、軽量な機械学習アルゴリズムによる最適化を組み合わせることで、計算コストを抑えつつ高精度なハイライト推定を実現しています。

## 構成概要
1. **Phase 1: 特徴量抽出**
Qwen2.5-Omniを用いて、音声データから直接VAD（Valence, Arousal, Dominance）スコアを時系列配列として抽出します。
2. **Phase 2: ハイライト確率推定**
抽出されたVAD配列と、人間の主観評価による正解ラベルを用いて、ランダムフォレストを学習させます。

## ファイル説明
* `VAD_score_matrix.ipynb`
音声ファイルを読み込み、指定した秒数（チャンク）ごとに分割し、Qwenモデルに入力してVADスコアを推論・抽出するスクリプトです。結果はCSV形式（Time, V, A, D）で出力されます。
* `highlight_estimated_by_VAD.ipynb`
出力されたVADスコアのCSVと、正解ラベルのCSVをマージし、ランダムフォレストを用いてハイライト確率を学習・推論するスクリプトです。

## 本アプローチの技術的な特徴
* **非線形特徴量エンジニアリング**
生のValenceスコア$V$に加え、中立状態からの乖離を示す絶対値演算$|V-5|$を新たな特徴量として試験的に導入しています。これにより、極端な喜びと極端な怒りや悲しみの両方をハイライトとして捉える非線形な関係性をモデルに学習させています。
* **音声分離の比較検証**
BGMやゲーム内SEを含む「通常音声（Mixed）」と、配信者の声のみを抽出した「分離音声（Isolated）」の2パターンのデータセットをフラグ一つで切り替えて比較検証できる設計となっています。
また、実証での音声分離は、Ultimated Vocal Remover(ver.5.6.1)において、UVR-MDX-NET-Voc_FTモデルを使用しています。
* **モデルの解釈性の担保**
Partial Dependence Plot（PDP）を用いて、AIが学習した各VADパラメータとハイライト確率の関係性を可視化し、モデルの推論根拠を視覚的に説明・証明可能です。

## 使い方
### Step 1: VADスコアの抽出
1. 対象の音声ファイル（.wav）をGoogle Drive上で `inputs/` フォルダに配置します。
2. `VAD_score_matrix.ipynb` を実行し、`outputs/` フォルダにVADスコアのCSVを生成します。

### Step 2: モデルの学習と推論
1. 人間の主観評価から作成した正解ラベルCSV（Time, Highlight_Prob）を `labels/` フォルダに配置します。
2. `highlight_estimated_by_VAD.ipynb` の設定スイッチ（`USE_VOCAL_ISOLATED`）を選択し、セルを順番に実行します。
3. 学習完了後、テストデータに対する予測結果の時系列グラフ表示と、`predictions/` フォルダへの予測CSVの保存が自動で行われます。

## 注意事項 (Note)
